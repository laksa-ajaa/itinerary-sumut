# Hero Banner Assets

This directory contains hero banner images for the Itinerary Sumut application.

## Files:
- `hero-banner.svg` - Vector-based hero banner featuring Sumatera Utara landscape with mountains, lake, palm trees, and traditional house
- `hero.png` - Existing hero image

## Usage:
To use this hero banner in your views, reference it like this:
```php
style="background-image: url('{{ asset('assets/img/hero-banner.svg') }}')"
```

## Notes:
- The SVG is scalable and will look good on all screen sizes
- Colors are optimized for overlay text readability
- Features traditional Indonesian elements representing North Sumatra
